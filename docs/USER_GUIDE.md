# MyFi User Guide

MyFi is a unified finance platform for Vietnamese stock market analysis and portfolio management.

## Getting Started

### 1. Create an Account

Navigate to the login page and click "Register". Provide a username (min 3 characters), password (min 8 characters), and optional email.

### 2. Navigate the Dashboard

After login, you land on the **Overview** tab. Use the sidebar to switch between tabs:

| Tab | Purpose |
|-----|---------|
| Overview | Market summary, watchlist, quick stats |
| Markets | Detailed market data, charts, company info |
| Portfolio | Your holdings, transactions, performance |
| Screener | Filter stocks by fundamental metrics |
| Sectors | Sector performance and breakdown |
| Comparison | Compare multiple stocks side-by-side |
| Goals | Financial goal planning and tracking |
| Signals | AI-generated trading signals |

### 3. Set Your Preferences

- **Theme**: Toggle between light and dark mode via the header
- **Language**: Switch between Vietnamese (vi-VN) and English (en-US) using the language selector

## Market Charts

The interactive chart supports candlestick and volume visualization powered by TradingView Lightweight Charts.

### Features

- **Candlestick chart** with OHLCV data
- **Drawing tools**: Trendlines, horizontal lines, Fibonacci retracements
- **Technical indicators**: SMA, EMA, RSI, MACD, Bollinger Bands, and more
- **Time intervals**: 1D, 1W, 1M, and custom ranges
- **Crosshair** with price/time tooltip
